# BD2 Shortcuts v1.3

Hai file song song:

```text
SkillTextTable_EN.json
```

File game đang đọc.

```text
SkillTextTable_EN.raw.json
```

File làm việc/copy song song; game không đọc.

## Cài đặt

```bash
cd "$HOME/Downloads/BD2Shortcuts_v1.3"
chmod +x install.sh
./install.sh
source "$HOME/.bashrc"
```

## Quy trình

```text
bd2clear

bd2pull
```
